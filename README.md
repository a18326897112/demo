# Demo 项目

## 🐍 贪吃蛇小游戏

使用 Python 内置的 `tkinter` 库实现，无需安装额外依赖。

### 运行方法

```bash
python snake_game.py
```

> 需要 Python 3.x 环境

### 游戏操作

| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| P | 暂停 / 继续 |
| R | 重新开始 |

### 游戏规则
- 吃到红色食物得 10 分
- 撞到自身则游戏结束
- 穿越边界会从对侧出现
